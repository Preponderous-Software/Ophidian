# Ophidian
This game allows you to control an ever-increasingly growing ophidian in a virtual environment. 

## Controls
Key | Action
------------ | -------------
space | pause/unpause
w | move up
a | move left
s | move down
d | move right
f11 | fullscreen
l | toggle tick speed limit
r | restart
q | quit
